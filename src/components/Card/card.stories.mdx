# Card

A card need not have any specially-structured content. A plain card might simply contain some text:

<Story name="Basic">
  	<Card>Test</Card>
</Story>


On the other hand, a card may contain complex content:

<Story name="Complex">
	<Card>
		<Header slot="header">Test</Header>
		<Content><p>You need to perform some actions here.</p></Content>
		<Footer><button slot="footer">Done!</button></Footer>
	</Card>
</Story>


Cards can have "underlays":
